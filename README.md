# z3pta
PTA Solver utilizing Z3 SMT solver

* Z3 is required for the solver. Install Z3 from [here](https://github.com/Z3Prover/z3).
  * Visual studio does not recognize Z3 libraries automatically. Explicitly point to the library folder in the solver file.
  * There is no stable pip installer for Z3 at the moment.
* Python 3 and numpy are required.
